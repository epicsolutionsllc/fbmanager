# fbmanager
Better alternative to Facebook Business Dashboard, powered by Vercel Serverless Functions and Vue 3

## Project setup
```bash
$ npm install
```

### Compiles and hot-reloads for development
```bash
$ vercel dev
```

### Push to production
```bash
$ vercel --prod
```

### Lints and fixes files
```bash
$ npm run lint
```
